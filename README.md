# MoneyOS 手机模拟器

一个无需安装依赖的手机模拟器原型。

在线试玩：https://wubugui.github.io/huaqianmoniqi/

打开 `index.html` 即可游玩，也可以部署到 GitHub Pages。核心循环是：

- 初始只有点钞、电话、短信和应用商店。
- 点钞 App 全屏上滑数钱，大拇指从下往上一划才会数走一张钱；点多少获得多少，钱存在点钞 App 里。
- 点钞余额只能作为现金源头，不能直接消费；唯一例外是第一次办手机号时自动扣 `¥50` 充入话费。
- 所有新 App 都必须从应用商店下载。
- 下载营业厅后购买手机号，并办理 `¥19 不够用套餐` 或 `¥99 尊贵套餐`，手机才真正入网。
- 游戏按办业务/交易推进日期，30 天进入新月份；套餐到期后必须手动续费，否则除了点钞、应用商店和营业厅，其他 App 都会停机。
- 下载银行 App 后用手机号办理银行卡；外卖、购物、炒股等生活消费都从默认银行卡扣款。
- 点钞余额可以转入银行卡，但必须输入正确卡号；输错钱会消失。
- 银行卡号可以复制，再粘贴到转账表单。
- 钱包、账单、外卖、购物、旅行、汽车、房产、缴费、娱乐、炒股、出行、本地生活、医疗、保险、信用、订阅、快递、闲置、政务、教育、美业、宠物、充值、社交、租赁、办公、家装、母婴、票务、跨境、法律、求职、活动、安防等 App 都能花钱。
- 所有消费目录先进入确认付款面板，再扣钱；面板会显示商户、订单号、实名/地址资料、履约方式、平台服务费提示、押金可退额和支付限制。
- 应用商店本身也能付费下载、买内购、开云备份、买家庭共享和打赏开发者。
- 银行 App 支持账户短信通知、补卡、跨行手续费、征信报告、保险箱、大额存单和按揭评估服务。
- 支持联系人转账、红包/群红包、扫码付款、商家收款码、提现手续费、押金预授权、外卖下单、电商物流、购物售后退款、机酒门票、签证、买车养车、租房买房、生活缴费、会员/游戏充值、投资持仓、打车、保洁搬家、问诊买药、保单、自动续费、寄件、二手担保交易、教育缴费、美业预约、宠物消费、社交礼物、设备租赁、办公 SaaS、家装分期、母婴托育、票务退改签、跨境支付和政务缴费。
- 消费页支持切换银行卡/信用额度付款；转账、投资、房产、部分政务税费和跨境汇款会按真实限制拒绝信用支付。
- 法律、公证、诉讼、仲裁、求职背调、猎头意向金、婚宴定金、活动保险、智能门锁、监控云存储、隐私报告、数据恢复和家庭报警押金等高频支出已接入账单体系。
- 外卖 App 支持收货地址、配送方式、优惠券、骑手小费、餐具、隐私号、发票和备注，付款前会按真实费用结构重新计算实付金额。
- 外卖、购物、旅行、汽车、房产、缴费、娱乐和钱包继续补齐到店团购、订座押金、延保、会员年卡、改签差价、行李额、年检代办、锁房定金、宽带电视、校园卡、亲情卡和自动扣费授权等高频花钱入口。
- 打电话和发短信会实际扣话费余额，营业厅充值不再只是摆设。
- 车房首付会生成车贷/房贷合同，银行、汽车、房产 App 都能查看并按期还月供。
- 信用 App 支持开通额度、先用后付、现金借款到账、分期账单和银行卡还款。
- 每个消费 App 支持按业务标签筛选，付款项会显示押金、月供、续费、保单、持仓或不可撤回等真实提示。
- 所有银行卡消费会写入支付流水，部分消费会生成带订单号、商户、履约信息、服务费提示和发票资料的订单、行程、资产、保单、订阅、分期、贷款、押金或缴费记录。
- 订阅可手动续费或取消，押金/预授权可退回，信用分期可按期还款。
- 账单 App 会汇总银行卡余额、累计消费、本月固定支出、押金、信用和长期月供，并支持一键支付本月订阅、贷款、信用分期和信用最低还款。
- 开发清单位于 `SPENDING_APP_CHECKLIST.md`，记录每轮当前完善的 App、完成度和下一轮指针。

素材位于 `assets/sprites/`，当前是 `weird3` 版本：成人刻意模仿儿童画的荒诞无厘头风格，物件带怪脸和假幼稚符号。
